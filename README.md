# PHPCrystal Web application framework

PHPCrystal is a modular full-stack Web application framework based on an event-driven
model and several components of such well-known frameworks as Symfony and Zend.
In addition, it ships with Twig and Doctrine as its default template engine and
ORM/DBAL library respectively.

The goal of the framework is to provide Web developers with a tool allowing to
implement customer's requirements in a clear and concise way.
